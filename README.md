## 📓 简介
这是一个以原版生存为主要玩法的原版边界生存系列。从拥窄的区块开局，通过智慧与勇气获得资源，用自己的双手开放世界。

## 📗 游前须知
本系列完全可以用原版生存四个字完整概括。但在本系列中，对于资源的获取，却需要足够深入的思考。您可以将本系列当作益智游戏，但如果不是很了解原版特性内容，可能无法拥有良好的游戏体验。

这并非是一个轻松简单的生存，在这里，你需要拥有足够的特性理解，即使是正在编写这段文字的我，也不敢肯定完全熟知本系列的所有特性技巧。

这里没有任何目标，如果说需要一些挑战，那么可以将等级视为分数，而你是否又有信心，达成自己定下的目标，坚持到多少级弃档呢？本系列每一张地图的体验均不同，如果做好了准备，就开始你的挑战吧！

⚠ 1.17-1.19已停止更新，本系列仅更新最新Minecraft版本。旧版本不再维护，且不推荐游玩旧版本。本文档介绍的所有内容均为最新版本(1.20.4)内容。

## 🌊 核心特性

#### 📕 科普：什么是世界边界
世界边界是Minecraft1.8加入的特性，没错，它是原版中就有的。它是一堵淡蓝色透明的动态斜线光墙。边界拥有三种状态，常规状态颜色为蓝色、如果正在收缩则为红色、正在扩展则为绿色。你无法越出边界、无法操作边界外的方块。边界外面不会自然生成新的生物。边界外面的工作方块、部分红石元件无法运行。边界外面没有空气，如果离开边界一定范围，将会因缺氧持续受到窒息伤害。

#### 🔵 边界等级
边界等级即为边界大小，可通过消耗经验进行边界升级。

#### 💠 边界模式

**🔵 禁锢模式**
正常模式

**🔴 昼夜模式**
在昼夜模式晚上的第一分钟，边界将会向外逐渐扩展一倍，最多向外扩展100格。
在昼夜模式晚上的最后一分钟，边界将会逐渐回到应有的大小。

虽然昼夜模式的夜晚看似美好，但是潜在的黑暗却是危机四伏：在昼夜模式的夜晚，刷怪上限与单次刷怪数量将会提高。且怪物将无视光照等级刷新。

## 🗺️ 地图库

❗ **1.18版本之后不会再出现浮岛和洞穴类型的地图。**

**原版型**<br/>
原版型地图为默认世界类型。与常规原版世界没有任何区别。

**浮岛型 (仅1.17)**<br/>
浮岛型地图是由岛屿组成的世界，地形生成机制会令它们产生更多特性与趣味，同时也比原版型地图资源获取难度更高，对于低配玩家会更加流畅，且地形对于建筑玩家较为友好，多个岛屿多倍发展，让每个岛屿都留下你的足迹。

**洞穴型 (仅1.17)**<br/>
洞穴型地图刷怪量极高！且前期生存条件过于艰难，需要拥有充足的原版知识。该系地图难度较大，推荐红石生电系老原版玩家挑战。

**瀚海型 (仅1.17)**<br/>
瀚海型地图为无尽的海洋，整个世界只包含海洋群系，是本系列一周年特殊类型。

## 📖 常见问题
* **功能牌被破坏**<br/>
执行下方指令获取 (需创造模式放置)
```/function border_survival:get_sign```

* **进入末地掉入虚空？**<br/>
边界等级不足，边界没有扩展到主世界的X:0 Z:0坐标，如果正常找到要塞进入末地不会出现此情况，所以请不要自行拿末地传送门框架。

* **进入地狱门一直卡在世界加载中？**<br/>
如遇到此情况是地图本身问题，请前往issues提交地图名称通知修复。

* **为何不刷怪？** <br/>
请检查是否为和平模式，如果遇到地图默认为和平模式的情况，请提供地图名称通知修复。<br/>
因边界外部不会刷怪，如果边界等级小于10级，刷怪效率会较为缓慢。
	
* **为何进入游戏没加载区块？**<br/>
这是 **[流畅加载]smoothboot** 模组的功能，他可以使区块加载时游戏不会卡顿。等待片刻即可。

* **为何optifine已开启纹理衔接但是我的玻璃就是不衔接？**<br/>
请调整你的资源包排序，因部分模组含内置的资源包覆盖了原版的资源包导致纹理衔接功能失效，请保持最底下的三个资源包的排序如下图所示：<br/>

## 📖 注意事项
* **游戏设置**<br/>
不建议开启死亡不掉落，会大幅降低游戏难度。<br/>
不要开启常用的原版防苦力怕规则指令，开启之后村民不会捡起物品。<br/>
需要防爆效果请使用地毯模组的真防爆指令：
```/carpet setDefault explosionNoBlockDamage true```

* **环境与配置**<br/>
❗ 本系列包括1.17在内的所有版本均需要至少**Java17**以上才能运行。<br/>
无需关心你的CPU与GPU性能，本系列大多数为优化工具辅助模组，而且本身就是原版生存，只要对应版本的原版生存能流畅游玩就行。<br/>
对于内存，根据具体情况分配。一般2048M-3072M足够使用。如果你调的视距比较大，比如16,20,24，那还是建议多分配一点。

* **模组搭配**<br/>
除了标注了核心标签的模组，其他模组均可以随意增删。他们不影响游戏核心。例如可自行添加replay、投影等工具模组。<br/>
温馨提示：本系列为原版生存，请不要随意添加含新物品的玩法型、破坏原版机制的功能模组，以不破坏本系列的主题与平衡性。

* **关于存档升级**<br/>
对于原版型、单群系型地图可进行版本升级，如1.18升级至1.19。<br/>
对于1.17所有地图均不可进行升级。
