## 📓 简介
这是一个以原版生存为主要玩法的原版边界生存地图系列。从狭隘的空间启程，用智慧觅取资源，用勇气挑战昼夜，用经验拓展世界边界。适合对原版特性有一定了解的玩家挑战。

## 📗 游前须知
该系列并非常规整合包，它的本体由世界存档与核心模组组成，在整合包中仅加入了辅助性模组，需要安装整合包并导入专属世界存档才可进行游戏。

该系列与原版生存并无任何区别，无论是同样没有标准的目标与攻略，还是主要玩法。一百位玩家便会产生一百种独一无二的思路。

而相同且核心的，则是对原版特性以及该系列核心特性的利用技巧，空间规划，以及在某个世界中建造自己温馨的小窝。

虽与原版生存并无任何区别，但这并非是一个轻松、简单、愉快的生存，相反，它是一个休闲，硬核、益智的游戏。

该系列以原版生存为主要玩法，因该系列的第一位玩家喜欢世界边界的特效，自此一发不可收拾无法自拔，由此诞生了该系列的最初版本。

而因边界限制导致的或群系或结构的独有资源难以获取，于是一些资源产生了额外的获取方式，这些特性被称为核心特性。

当你遇到困难时，通过查阅Minecraft wiki所描述的原版特性和下列介绍的核心特性，或许可以有效帮助到你。

该系列存在固定的起点，但并不存在终点。每个世界的体验均不同，尽情发挥你所掌握的经验，留下你的足迹。

建议搭配自己喜欢的资源包进行游玩。

⚠ 本系列仅更新最新Minecraft版本。旧版本不再维护，且不推荐游玩旧版本。本文档介绍的所有内容均为最新版本(1.21.4)内容。

## 🌊 核心特性

#### 📕 科普：什么是世界边界
世界边界是Minecraft1.8加入的特性，没错，它是原版中就有的。它是一堵淡蓝色透明的动态斜线光墙。边界拥有三种状态，常规状态颜色为蓝色、如果正在收缩则为红色、正在扩展则为绿色。你无法越出边界、无法操作边界外的方块。边界外面不会自然生成新的生物。边界外面的工作方块、部分红石元件无法运行。边界外面没有空气，如果离开边界一定范围，将会因缺氧持续受到窒息伤害。

#### 🔵 边界等级
边界等级即为边界大小，可通过消耗经验进行边界升级。

#### 💠 边界模式
含有禁锢、昼夜两种模式，可在游戏中进行切换。

#### 🔵 禁锢模式
常规默认模式。

#### 🔴 昼夜模式
昼夜模式的白天十分祥和，与禁锢模式并无区别。

在昼夜模式的夜晚，刷怪上限与刷怪数量将会提高，且怪物将无视光照等级刷新。同时，若边界等级至少拥有100级，将额外扩展一倍空间。

#### 🌙 昼夜模式 — 永恒之夜
永恒之夜会在昼夜模式的夜晚，即将白昼时，且边界等级至少拥有100级时降临。之后将在度过0-4个游戏日后再次降临。<br/>
永恒之夜降临后，世界出生点周围将会出现异界的永夜骑士。<br/>
受永夜骑士的影响，时间会被禁锢在他们降临的那一刻，并且世界将陷入永恒的雷暴之中。<br/>
当永夜骑士的周围不存在玩家时，永夜骑士将持续恢复生命值。<br/>
当永夜骑士受到伤害时，将干扰世界生成更多怪物。<br/>
永夜骑士的气息会引起附近怪物的仇恨，但怪物们仍然会优先攻击玩家。<br/>
只有将所有降临的永夜骑士全部击败，才可解除其对世界的影响。

## 📖 注意事项
* **游戏规则**<br/>
❗ 即使开启死亡不掉落，死亡后也会掉落经验。<br/>
❗ 不推荐关闭生物破坏性行为，下面的指令一般被用于苦力怕防爆，但该指令会导致村民无法拾取物品、猪灵无法拾取金锭、怪物无法捡起盔甲工具。<br/>

✅ 如若需要阻止末影人搬土，推荐使用以下指令：<br/>

✅ 如若需要恶魂火球防爆，推荐使用以下指令：<br/>

* **环境与配置**<br/>
❗ 本系列需要至少**Java21**以上才能运行。<br/>
本系列大多数为优化辅助模组，而且本身就是原版生存。只要您的计算机对应版本的原版生存能够流畅游玩即可。<br/>
对于内存，根据具体情况分配。一般2048M-3072M足够使用。如果您视距拉的比较大，比如16,20,24，那还是建议多分配一点。

* **模组搭配**<br/>
在整合包mods文件夹中，被标注了[核心]标签的模组不可删除，他们被删除不会损坏存档，但可能导致存档无法正常发展或游戏无法启动。<br/>
其余模组可随意增加或删除，例如replay、REI等。自行添加的模组建议为其标注[自加]等前缀，以方便更新时精准替换模组。
❗ 不建议添加含有新物品方块的功能性模组。(主世界以外的维度模组除外)

## 📖 常见问题
* **功能牌被破坏** <br/>
执行下方指令获取 (需创造模式放置)
```/function border_survival:get_sign```

* **功能牌右键没反应？** <br/>
这是由于模组没有被加载导致的，请检查整合包是否安装正确，且启动的游戏版本名是否为整合包所安装的版本名。

* **指令无法使用？** <br/>
绝大多数指令都需要作弊权限才可使用。在单人模式中，可通过点按键盘左上角的 **ESC** 键呼出菜单之后，使用鼠标点击 **对局域网开放按钮**，将 **允许作弊: 关** 更改为 **允许作弊: 开**，之后使用鼠标点击**创建局域网世界**按钮便可更改作弊权限。<br/>
在服务端环境下，需要通过服务端控制台使用 /op <玩家名称> 指令给予玩家作弊权限才可使用。

* **进入末地直接死亡？** <br/>
因边界等级不足，边界未扩展到末地黑曜石平台坐标，传送门将玩家传送到末地主岛后，距离边界过远，故而窒息而死。请将边界等级至少提升至200级后，再尝试进入末地。

* **为何不刷怪？** <br/>
请检查是否为和平模式，如果遇到地图默认为和平模式的情况，请前往issues提交地图名称通知修复。<br/>
因边界外部不会刷怪，如果边界等级小于10级，刷怪效率会较为缓慢。

* **放置方块延迟？区块加载卡顿？活塞动画不流畅？** <br/>
这可能是动态光源导致的，请尝试关闭动态光源：**ESC->模组->RyoamicLights->右侧配置->动态光源: 关**，或仅关闭方块实体的动态光源。

* **搭建的服务端覆盖整合包模组后无法正常启动？** <br/>
服务端需删除 **[更高级联机设置]mcwifipnp** 模组才可正常运行。

* **搭建的服务端默认没有开启地毯carpet模组的任何功能？** <br/>
服务端的地毯模组需要单独为存档配置：请将 **config/carpet/default_carpet.conf** 文件复制到存档根目录下，并将该文件重命名为 **carpet.conf**，之后重启服务端即可。

## 🎓 模组技巧
本系列整合包中对部分模组按键进行了配置，在游戏中可通过以下操作提升游戏体验：

* 使用 鼠标右键 点击村民交易项进行一键交易
* 使用 R 整理背包/箱子
* 使用 Shift 预览潜影盒内容
* 使用 G、J、↑、↓ 调整Gamma值
* 使用 F6 开启摄像机模式 (tweakeroo灵魂出窍)
* 使用 F7 显示光照等级
* 使用 F8 查看史莱姆区块
* 使用 F9 查看结构区块
* 使用 [ 开关灵活放置方块。
* 使用 ] 开关方块快速放置。
* 使用 右shift 开关伪潜行。
* 使用 鼠标滚轮 移动单个物品
* 使用 shift + 左键 移动一组物品
* 使用 shift + 左键按住 + 拖动 快速移动多组物品
* 使用 alt + 左键 移动所有相同物品
* 使用 shift + alt + 左键 移动所有物品
* 使用 Q + shift + 左键按住 + 拖动 快速丢出多组物品

## 📔 使用方式

#### 安装
该系列分为整合包和地图包，其中，以 **BorderSurvival** 命名的为整合包，以 **边界生存-XXXX** 命名的为地图包。

整合包由[HMCL](https://hmcl.huangyuhui.net/download/)启动器打包导出，请使用支持安装整合包的启动器，将整合包拖拽到启动器中即可进行安装。

在整合包安装成功后，使用HMCL启动器的版本管理->世界/数据包管理->添加存档功能导入地图压缩包，或使用Bandizip的自动解压功能，将存档压缩包放在 **.minecraft/versions/版本名称/saves** 目录下进行自动解压。

#### 更新
简单的方式：(以下两种方法会导致模组配置重置)
- 快捷更新：通过[HMCL](https://hmcl.huangyuhui.net/download/)启动器的更新整合包功能进行更新。
- 重装整合包：将存档备份之后，重新安装新版本整合包，再将备份的存档迁移到新版本中。

更精准的方式：(推荐的方式，需要对游戏目录结构有一定了解)
- 重装模组：删除当前游戏版本mods文件夹中整合包自带的模组。下载新版整合包，将压缩包中overrides/mods目录下的模组解压到当前游戏版本的mods目录中。

数据包更新：
- 部分含有特殊玩法的地图中含有数据包，若需要更新这些功能，请下载新版地图包。然后删除掉当前存档中的datapacks目录。之后在确保删除掉旧版本数据包后，将下载的新版地图包中的datapacks文件夹解压到当前存档中。<br/>
**❗ 一定要先删除再放入，切勿直接覆盖，直接覆盖将会导致数据紊乱**<br/>
❗ **切勿直接覆盖整个存档！若区块文件被覆盖将无法挽回！请在进行操作之前备份好存档，并清楚你正在做什么。**
