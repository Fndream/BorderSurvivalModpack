## 📓 简介
这是一个以原版生存为主要玩法的原版边界生存系列。

## 📗 游前须知
本系列完全可以用原版生存四个字完整概括。但在本系列中，对于资源的获取，却需要足够的思考。

你需要以原版特性为主、并结合地毯(carpet mod)特性、整合包核心特性、甚至是地图群系特性，通过它们来获取因边界限制所稀缺或无法获得的资源的种子。如果不是很了解原版特性，可能无法拥有良好的游戏体验。

这并非是一个轻松简单的生存，你需要拥有足够的原版生存经验，即使是正在编写这段文字的作者，也不敢肯定完全熟知本系列的所有特性技巧。但相同且核心的是，建造一个温馨的小窝~

## 🌊 核心特性

#### 📕 科普：什么是世界边界
世界边界是Minecraft1.8加入的特性，没错，它是原版中就有的。它是一堵淡蓝色透明的动态斜线光墙。边界拥有三种状态，常规状态颜色为蓝色、如果正在收缩则为红色、正在扩展则为绿色。你无法越出边界、无法操作边界外的方块。边界外面不会自然生成新的生物。边界外面的工作方块、部分红石元件无法运行。边界外面没有空气，如果离开边界一定范围，将会因缺氧持续受到窒息伤害。

#### 🔵 边界等级
边界等级即为边界大小，可通过消耗经验进行边界升级。

#### 💠 边界模式
含有禁锢、昼夜两种模式。

#### 🔵 禁锢模式
常规默认模式，无特殊特性。

#### 🔴 昼夜模式
昼夜模式的白天与禁锢模式并无区别。<br/>
在昼夜模式的夜晚，刷怪上限与刷怪数量将会提高，且怪物将无视光照等级刷新。同时，若边界至少拥有100级，将会额外扩展一倍空间。

#### 🌙 昼夜模式 — 永恒之夜
永恒之夜会在昼夜模式的夜晚，即将白昼时，且边界等级至少拥有100级时触发。每个游戏日有1/7概率触发，若连续六日未触发，在第七日必定触发。

永恒之夜降临后，世界出生点周围将会出现异界的永夜骑士。受永夜骑士的影响，时间会被禁锢在他们降临的那一刻，并且世界将陷入永恒的雷暴之中。

当永夜骑士周围不存在玩家时，永夜骑士会持续恢复生命值。

永夜骑士的气息会拉取附近怪物的仇恨，但怪物们仍然会优先攻击玩家。只有将所有降临的永夜骑士全部击败，才可解除其对世界的影响。

## 📖 注意事项
* **游戏设置**<br/>
即使开启死亡不掉落，死亡后也会掉落经验。<br/>
若村民不会捡起物品，是因为开启了防爆。

* **环境与配置**<br/>
❗ 本系列需要至少**Java21**以上才能运行。<br/>
本系列大多数为优化辅助模组，而且本身就是原版生存。只要您的计算机对应版本的原版生存能够流畅游玩即可。<br/>
对于内存，根据具体情况分配。一般2048M-3072M足够使用。如果您视距拉的比较大，比如16,20,24，那还是建议多分配一点。

* **模组搭配**<br/>
除了标注了**核心**标签的模组，其他模组均可以随意增删，它们不影响游戏的核心体验。例如可自行添加replay、投影等工具模组。<br/>
不建议添加含有新物品方块的模组。

## 📖 常见问题
* **功能牌被破坏**<br/>
执行下方指令获取 (需创造模式放置)
```/function border_survival:get_sign```

* **进入末地直接死亡？**<br/>
因边界没有到达主世界的 X:0 Z:0 坐标。传送门将玩家传送到末地主岛后，距离边界过远，故而窒息而死。请先将边界扩展到主世界的 X:0 Z:0 坐标后，再尝试进入末地。

* **为何不刷怪？** <br/>
请检查是否为和平模式，如果遇到地图默认为和平模式的情况，请前往issues提交地图名称通知修复。<br/>
因边界外部不会刷怪，如果边界等级小于10级，刷怪效率会较为缓慢。<br/>
整合包中含有SpawnAnimations模组，该模组会使怪物隐身，在玩家靠近时才会伴随动画出现。受该模组影响，使用tweakeroo的灵魂出窍无法看到远处怪物，旁观者模式可以看到隐形的怪物。可通过来自地毯模组的如下指令查看当前生物数量/生物上限情况。输入该指令后，需按住TAB键在屏幕上方查看。

## 🎓 模组技巧
本系列整合包中对部分模组按键进行了配置，在游戏中可通过以下操作提升游戏体验：

* 使用 鼠标右键 进行村民一键交易
* 使用 R 整理背包/箱子
* 使用 Shift 预览潜影盒内容
* 使用 G、J、↑、↓ 调整Gamma值
* 使用 F6 开启摄像机模式 (tweakeroo灵魂出窍)
* 使用 F7 显示光照等级
* 使用 F8 查看史莱姆区块
* 使用 F9 查看结构区块
* 使用 鼠标滚轮 移动单个物品
* 使用 shift + 左键 移动一组物品
* 使用 shift + 左键按住 + 拖动 快速移动多组物品
* 使用 alt + 左键 移动所有相同物品
* 使用 shift + alt + 左键 移动所有物品
* 使用 Q + shift + 左键按住 + 拖动 快速丢出多组物品

## 📔 使用方式

#### 安装
整合包由[HMCL](https://hmcl.huangyuhui.net/download/)启动器打包导出，请使用支持安装整合包的启动器导入安装。

#### 更新
简单的方式：
1. 通过[HMCL](https://hmcl.huangyuhui.net/download/)启动器的更新整合包功能进行更新。
2. 将存档备份之后，重新安装新版本整合包，再将备份的存档迁移到新版本中。

其他方式：(需要对游戏目录结构有一定了解)
1. 删除当前游戏版本mods文件夹中整合包自带的模组。下载新版整合包，将压缩包中overrides/mods目录下的模组解压到当前游戏版本的mods目录中。
2. 部分含有特殊玩法的地图中含有数据包，若要更新这些功能，请下载新版地图包，之后删除掉当前存档中datapacks目录下的原有数据包，在确保删除掉旧版本数据包后，将下载的新版地图包中datapack目录下的数据包解压到当前存档中。
❗ 切勿直接覆盖整个存档！若区块文件被覆盖将无法挽回！请在进行操作之前备份好存档，并清楚你正在做什么。
